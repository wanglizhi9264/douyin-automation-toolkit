# 抖音收藏与下载工具

Chrome MV3 插件版本，用浏览器自己的抖音登录态运行，后续逐步替代 Playwright 脚本。

## 当前能力

- 在 `www.douyin.com` 页面右侧注入本地侧边栏。
- 打开侧栏后快速读取抖音官方喜欢和收藏总数。
- 分页检查并下载喜欢、收藏视频，图文和已下载作品自动跳过。
- 记住用户授权文件夹，以文件夹记录为准继续下载。
- 复用列表中的最高画质候选，缺失或过期时再获取作品详情。
- 下载视频、可选封面、单条 manifest、本地数据库、状态与日志。
- 记录列表、详情、限流、媒体传输、磁盘写入和 UI 各阶段耗时。
- 导入后端脚本进度，并保留继续收藏和审计能力。

## 运行说明

- 首次使用先点“选择文件夹”；后续任务优先复用该目录。
- “下载喜欢视频”和“下载收藏视频”都从第一页重新核对，以文件夹记录跳过已下载项。
- “继续下载”恢复上次作用域，不依赖可能过期的远端游标。
- 最高画质最多尝试 4 个去重候选，失败时按顺序回退。
- 任务结束后查看 `data/.appdata/performance-summary.json` 判断慢在接口、主动等待、网络还是磁盘。
- 遇到限流、验证码、登录失效或连续 8 条失败会停止并保留诊断。

## 本地安装

1. 打开 Chrome：`chrome://extensions`
2. 开启开发者模式。
3. 点击“加载已解压的扩展程序”。
4. 选择本仓库的 `plugin` 目录。
5. 打开 `https://www.douyin.com/`，登录后点击扩展图标显示侧边栏。

## 开发命令
